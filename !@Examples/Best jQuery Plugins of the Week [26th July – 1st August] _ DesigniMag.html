Best jQuery Plugins of the Week [26th July – 1st August]

  • August 2, 2015
  • /
  • Coding
  • /

jQuery plugins have got the deserving place in the filed of designing and development for the ease of use, the end result, the freedom from all the hassle and much more. Each week we list down some of the really amazing, functionally smart jQuery plugins for you all the check out and today we are here with yet another compilation.

From DateDropper to inputmask, enllax.js, imgCheckbox, there are some of the really interesting plugins for you all to use them in your different design and development projects. Have a look!

Best jQuery Plugins of the Week [21st June – 27th June]

1. DateDropper

datedropper is a jQuery dates plugin that ensures to provide a quick and easy way to manage dates for input fields. Easy-to-use, the plugin supports 16 different language formats.

Demo | Download

2. yacal

Yacal is a lightweight jQuery calendar plugin that is quite easy to configure and use.The plugin comes with default template and can be tuned up as per your requirement.

Demo | Download

3. wdImageStax

This is a plugin for scattered stacked image gallery.

Demo | Download

4. imgCheckbox

This is a jQuery plugin that makes image tags checkable. It comes without any parameters on any jQuery collection containing img elements

Demo | Download

5. fullBleed

fullBeed is a jQuery plugin that scales an image to take up the entire browser window.

Download

6. enllax.js

enlaxx is an ultra-lightweight and easy-to-use jQuery plugin that can be used for applying parallax scrolling effect to any scrolling element.

Demo | Download

7. inputmask

This is a jquery plugin that create an input mask thereby helping user with the input by ensuring a predefined format.

Demo | Download

8. NumPad

This is one flexible touch-optimized numeric keypad for web applications that are based on jQuery.The plugin helps in filling inputs, password fields, general div-elements or entire table columns.

Demo | Download

9. Secretnav

Secretnav is a jQuery plugin that reveals a menu at the top or left of a web application with superb 3d effect.

Demo | Download

That’s all for this week. Hope you like the list of best jQuery plugins of the week and comment to share your recently created plugins with us.
